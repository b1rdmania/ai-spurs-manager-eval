Each manager is evaluated using a Unified Final Scoring System combining two models:

Final Score = (Peer Score x 10 x 0.40) + (Spurs-Fit Total x 0.60)

Peer Analysis (40%)

Nine categories scored 0-10 against the candidate pool: Pressing Intensity, Attacking Quality, Defensive Solidity, Big Game Performance, Youth Development, Squad Health, Transfer Acumen, Media Stability, and Stakeholder Alignment. Each KPI is percentile-normalized within the peer group.

Spurs-Fit Model (60%)

Composed of two sub-models:

  • Fit Index (60%): Four Spurs-specific benchmarks — front-foot play, youth pathway, talent inflation, and big-game mentality — scored using continuous sigmoid curves (no binary pass/fail).
  • Potential Index (40%): Forward-looking assessment using Gaussian age curve (peak at 43), season-over-season trend, resource leverage (overperformance vs budget), and temperament (derived from observable facts: touchline bans, board disputes, contract stability).
Spurs-Fit = (Fit Index x 0.60) + (Potential Index x 0.40)
